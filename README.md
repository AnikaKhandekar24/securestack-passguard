# SecureStack by Anika: PassGuard

PassGuard is a beginner-friendly password strength checker built with plain HTML,
CSS, and JavaScript. It runs entirely in the browser and does not store or send
the password being checked.

## Features

- Live password score from 0 to 100
- Five-level strength meter
- Length, letter case, number, and symbol checks
- Detection for common passwords, sequences, and repeated characters
- Simple educational time-to-crack estimate
- Personalized improvement feedback
- Password visibility control
- Responsive and accessible interface
- Security tips covering long passwords, unique passwords, and MFA

## Run the project

Open `index.html` in a modern browser. No installation or build command is
required.

## Files

- `index.html`: Page content and accessible structure
- `styles.css`: Responsive visual design
- `app.js`: Password analysis and interface updates

## Important note

The crack-time estimate is intentionally simplified. Real cracking speed depends
on the password hashing method, attacker resources, leaked-password lists, and
other factors. The estimate is educational and should not be treated as a
guarantee.
